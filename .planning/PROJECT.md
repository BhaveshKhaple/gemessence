# Project Brief: Gemessence

## Core Value
A blazing-fast storefront that sells handmade gemstone bracelets while demonstrating AI marketing techniques (Traffic, CTR, Automation, and Conversions).

## Target Audience
Customers seeking handmade gemstone bracelets, as well as stakeholders/evaluators for the "AI in Digital Marketing" class project.

## Project Scope (v1)

### Frontend & UI/UX
- Blazing-fast frontend framework (e.g., HTML/Tailwind/Vite or Next.js).
- Clean, modern layout mimicking the reference UI at `https://bling-display-corner.lovable.app`.
- Key sections: Hero section, product grid, and interactive cart sidebar.

### Data
- Products (names, descriptions, images) must be extracted from the Canva site `https://gemessencebracelet.my.canva.site/gem-essence-bracelet` into a JSON structure (`products.json`).

### AI & Marketing Integrations
- **AI Chatbot**: A chatbot providing customer support and mood-based gemstone recommendations.
- **Analytics**: GA4 placeholder integrated for tracking traffic and CTR.
- **Email Automation**: A promotional popup (e.g., "Subscribe for 10% off") that saves leads to a mock backend/JSON file.
- **Automated Agentic SEO**: Dynamic generation of meta tags and AI-generated image alt-text based on the product data.
